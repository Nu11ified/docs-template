# Welcome to the Docs

This is a sample documentation site built with Bun + ElysiaJS + React.

## Quick Start

Install dependencies and start the dev server:

```bash
bun install
bun run dev
```

## Features

- **Fast** — Built on Bun for maximum performance
- **Customizable** — Configure everything via YAML
- **Full-text Search** — Server-side search across all docs

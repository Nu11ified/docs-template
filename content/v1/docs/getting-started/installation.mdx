# Installation

## Prerequisites

- [Bun](https://bun.sh) v1.0 or later

## Steps

1. Clone the template
2. Install dependencies
3. Start the dev server

```bash
bunx degit org/docs-template my-docs
cd my-docs
bun install
bun run dev
```

## Configuration

Edit `site.yaml` to customize your documentation site.
